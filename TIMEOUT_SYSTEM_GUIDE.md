### 2. Condition Variable Pattern

```python

    """ƒê·ª£i v·ªõi condition variable - hi·ªáu qu·∫£ h∆°n polling"""

            if remaining <= 0:

            cv.wait(timeout=remaining)
            # Check condition after wakeup

                return True
```
